# Instalando o Git em seu desktop

## Windows

Para instalar o Git no Windows basta fazer o download da versão `standalone` ou `portable` de acordo com a [arquitetura do seu processador](https://support.microsoft.com/pt-br/windows/vers%C3%B5es-de-32-bits-e-64-bits-do-windows-perguntas-frequentes-c6ca9541-8dce-4d48-0415-94a3faa2e13d) clicando neste [link](https://git-scm.com/download/win/). Caso faça o download da versão `standalone` será preciso que execute o instalador e siga os passos demonstrados.

## MacOS

A instalação no MacOS também é muito simples. Basta seguir as instruções deste [link](https://git-scm.com/download/mac).

## Linux

Para distribuições baseadas em Debian/Ubuntu basta executar o comando abaixo em um terminal:

```bash
sudo apt-get install git
```

*Obs:* Para instalar em outras distribuições Linux confira a [documentação oficial](https://git-scm.com/download/linux) do Git.
